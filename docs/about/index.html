About Djalal Harouni (tixxdz)

This is a small bio (not a résumé) that is needed for the business side of things and clients’ requests.

Djalal Harouni (tixxdz) has been involved with computers since 1996, and has been professionally developing software since 2002. Djalal is an Open Source Software (OSS) developer and maintainer, with a focus on improving Linux security in general.

Djalal worked with several US and German companies around OSS and Linux ecosystems. He has contributed and helped to maintain popular OSS projects from systemd where he worked on the sandboxing environment and kernel interfaces, Linux kernel security, filesystems, container interfaces, Linux containers tools to kernel drivers and Linux embedded systems. He focuses now on the Cloud Native technologies and secure software deployment.

Djalal was an Nmap maintainer, worked on network scanning techniques, scripting engine and vulnerability scanning support. He also contributed to the Open Source telephony stack and development framework Ofono, and has many contributions to other OSS projects.
